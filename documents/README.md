# Папка документов

Просто положите сюда файлы (PDF, JPG, PNG) — сертификаты, декларации,
удостоверения качества и т.д.

Ничего больше делать не нужно:
- при каждом пуше в `main` GitHub Action сам просканирует эту папку
- сформирует список файлов (`manifest.json`)
- сайт (раздел «Документы») сам подхватит новые файлы и покажет их карточками

## Как называть файлы

Название карточки на сайте берётся из имени файла.
Используйте `-` или `_` вместо пробелов, они превратятся в пробелы:

```
deklaraciya-sootvetstviya.pdf   →  "Deklaraciya sootvetstviya"
udostoverenie-kachestva.jpg     →  "Udostoverenie kachestva"
```

Можно называть и по-русски — русские имена файлов тоже поддерживаются
(deklaraciya-sootvetstviya.pdf можно назвать декларация-соответствия.pdf).

## Что не трогать

Файлы `manifest.json` и `.gitkeep` в этой папке — служебные, их создаёт
и обновляет автоматика. Не удаляйте и не редактируйте их руками.
