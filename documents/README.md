# Папка документов

Просто положите сюда файлы (PDF, JPG, PNG) — сертификаты, декларации,
удостоверения качества и т.д. — прямо через файловый менеджер
хостинга (Eurobyte) или по FTP.

Больше ничего делать не нужно: файл `documents-list.php` в корне сайта
сам сканирует эту папку при каждой загрузке страницы и передаёт список
файлов в скрипт сайта. Никаких манифестов редактировать не надо.

## Как называть файлы

Название карточки на сайте берётся из имени файла.
Используйте `-` или `_` вместо пробелов, они превратятся в пробелы:

```
deklaraciya-sootvetstviya.pdf   →  "Deklaraciya sootvetstviya"
```

Можно называть и по-русски — русские имена файлов тоже поддерживаются.

## Поддерживаемые форматы

PDF, JPG, JPEG, PNG, WEBP. Файлы других форматов игнорируются.
